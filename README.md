# Delivery Dispatch Platform

배달대행 기사 앱과 관제 시스템을 단계적으로 개발하는 프로젝트입니다.

기사 앱은 React Native와 TypeScript를 기반으로 구성하며, 백그라운드 위치 추적처럼 Android 시스템과 직접 맞닿는 기능은 Kotlin으로 구현합니다.

## 개발 원칙

- 기능 단위로 이슈와 작업 브랜치를 관리합니다.
- 구현 후 타입 검사와 Android 빌드를 확인합니다.
- 운행 상태와 주문 상태가 유실되지 않도록 안정성을 우선합니다.
- 검증이 끝난 변경 사항만 기본 브랜치에 병합합니다.

## 예정 구성

- 기사용 Android 앱
- 지사용 관제 웹
- 상점 주문 접수 웹
- 주문·배차·위치·정산 서버
